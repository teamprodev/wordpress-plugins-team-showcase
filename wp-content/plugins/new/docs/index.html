Get started with Elfsight Team Showcase 1.3.1

Installation

1. Install Elfsight Team Showcase WordPress Plugin elfsight-team-showcase-cc.zip to your website. Don't know how? Please learn here.

2. Go to Team Showcase page in WordPress admin menu.

3. Create a new widget, adjust all settings and get a ready-made shortcode.

Use Team Showcase as a WordPress Shortcode



1. Create a new widget

You can use Elfsight Team Showcase as a WordPress shortcode. To do this, you have to create a new widget, set all options in the Elfsight editor and save it.

2. Copy-paste the shortcode

You will see the new widget's shortcode in the widgets list. You can copy-paste it into the required place of your website.

Example of the Elfsight Team Showcase shortcode:

[elfsight_team_showcase id="1"]

Use Team Showcase as a Gutenberg Block

You can use Elfsight Team Showcase as a block for Gutenberg editor. Just add it to your page during page building and select one of the available Team Showcase widgets from the list.

Use Team Showcase as a WordPress Widget

To use Elfsight Team Showcase as a WordPress widget, go to Appearance -> Widgets, and drag Elfsight Team Showcase WordPress widget to any widget section. Next, select one of available Team Showcase widgets from the list.

Use Team Showcase as a Visual Composer Element

You can use Elfsight Team Showcase as a Visual Composer element. Just add it to your page during page building and select one of the available Team Showcase widgets from the list.

Members: Website - Add the team member's personal website if necessary.

Members: Quote - Add the team member's motto that characterizes them in the best way. This quote will be displayed in the list layout, and with other layouts it will be displayed in the popup.
Members: Facebook - Add the person's Facebook account by pasting their Facebook username or URL.
Members: Twitter - Add the person's Twitter account by pasting their Twitter username or URL.
Members: LinkedIn - Add the person's LinkedIn account by pasting their LinkedIn username or URL.
Members: Instagram - Add the person's Instagram account by pasting their Instagram username or URL.
Members: YouTube - Add the person's YouTube channel by pasting their YouTube username or channel URL.
Enable groups false Activate this option to group your team members by departments, geographical location or any other criteria.
How to display groups Filters Choose the display mode for team member groups. List allows you to display every group one after another, and Filters allows you to filter all team members by the requied group.
Groups - Create your groups.
Widget Title Meet Our Team Add a title if necessary. It will be displayed above the list of team members.
